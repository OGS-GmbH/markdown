# Gettings started

## Installation

### Prerequisites

- Node.js version 18 or higher.
- A package manager: e.g. npm, pnpm, ...

::: code-group

```sh [npm]
$ npm add -D @ogs-gmbh/markdown
```

```sh [pnpm]
$ pnpm add -D @ogs-gmbh/markdown
```

```sh [yarn]
$ yarn add -D @ogs-gmbh/markdown
```

```sh [bun]
$ bun add -D @ogs-gmbh/markdown
```

:::

### Usage

We support all available elements in flavors:

- ✅ GitHub
- ✅ markdown-it
- ✅ Vitepress

Check the [builder reference](). Each element is shipped with an example.
